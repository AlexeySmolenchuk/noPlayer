Lightweight image inspector based on OpenImageIO and ImGui.

## Features
* Fast loading
* View AOVs in multi-channel and multi-part images
* View textures MIPs
* Pixel values inspector
* NaN and Inf detector
* Basic support for ACEScg from OCIO 2.0
* Shortcuts based (Listed by F1)

## Build
Compiles on Linux and Windows using vcpkg
